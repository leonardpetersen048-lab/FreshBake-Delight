FreshBake Delight Website
1. Introduction

For this project, I created a website called FreshBake Delight. The main idea behind the website is to showcase a bakery and the different products it offers, such as bread, cakes, and pastries. I wanted to design something simple, clean, and easy for users to understand.

The website was built using basic web development tools like HTML. My goal was to make sure the website looks good and is also easy to navigate for anyone visiting it.

2. Purpose of the Website

The purpose of this website is to present a bakery business online. It allows users to view different baked products and get an idea of what the bakery sells.

I designed the website in a way that:

Users can easily move between pages
Products are clearly displayed
Images make the site more attractive
3. Tools and Technologies Used

To build this project, I used:

HTML to create the structure of the website
Images to make the website visually appealing

I kept everything simple because this project focuses on the basics of web development.

4. Website Structure

The website consists of multiple pages, and each page has its own purpose.

Home Page

This is the first page users see. It gives a short introduction to the bakery and welcomes visitors.

Product Pages

I created different sections to display items like:

Bread
Cakes
Pastries

Each product includes an image and a name so users can easily identify it.

Navigation Bar

I added a navigation bar at the top of the pages. This helps users move between different sections of the website without getting lost.

5. Design Choices

When designing the website, I made a few important decisions:

I used simple colours so the website does not look too busy
I kept the layout clean and easy to read
I used images to make the website more attractive

I also tried to keep everything consistent across all pages so the site looks professional.

6. Use of Images

I included several images of bakery products such as bread, cakes, and muffins. These images help make the website more interesting and give users a better idea of the products.

I chose images that match the theme of the bakery and make the website look more realistic.

7. Challenges Faced

While working on this project, I faced a few challenges:

Getting the layout to look the same on all pages
Making sure images fit properly without stretching
Keeping the design simple but still attractive

I solved these problems by adjusting my CSS and testing the pages multiple times.

8. What I Learned

From this project, I learned:

How to structure a website using HTML
How to organize content in a clear way
The importance of planning before building a website

This project helped me understand how websites are created from scratch.

9. Conclusion

In conclusion, I successfully created a basic bakery website using HTML. The website meets its purpose by displaying products and allowing users to navigate easily.

If I had more time, I would improve the design further and maybe add more features like forms or interactivity using JavaScript.
